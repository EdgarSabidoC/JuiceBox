## Acceso al sitio web de Root The Box

Una vez el servicio este corriendo, se puede acceder al sitio web de Root The Box por medio del siguiente enlace:

`http://ctf.uady:8888`

O bien, utilizar el enlace por medio de la IP:

`172.16.200.11:8888`

De forma predeterminada el servicio de Root The Box corre en el puerto `8888`.
